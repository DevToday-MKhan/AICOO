## üìã Railway Deployment Steps

### Step 1: Create Railway Project

1. Go to [railway.app](https://railway.app)
2. Click "New Project"
3. Select "Deploy from GitHub repo"
4. Choose `DevToday-MKhan/AICOO`
5. Railway will automatically detect the Dockerfile

### Step 2: Add PostgreSQL Database

1. In your Railway project, click "+ New"
2. Select "Database" ‚Üí "PostgreSQL"
3. Railway will automatically create `DATABASE_URL` variable

### Step 3: Configure Environment Variables

Add these in Railway dashboard ‚Üí Variables:

```env
# Shopify Configuration
SHOPIFY_API_KEY=your_api_key_here
SHOPIFY_API_SECRET=your_api_secret_here
SHOPIFY_SCOPES=read_products,write_products,read_orders,write_orders,read_customers,write_customers,read_inventory,write_inventory
SHOPIFY_APP_URL=https://your-app.railway.app
SHOPIFY_API_VERSION=2024-10

# OpenAI
OPENAI_API_KEY=sk-your-openai-key-here
OPENAI_MODEL=gpt-4-turbo-preview

# Session
SESSION_SECRET=generate-random-32-char-string-here

# Database (auto-generated by Railway PostgreSQL)
DATABASE_URL=postgresql://... (Railway provides this)

# Environment
NODE_ENV=production
```

### Step 4: Get Shopify Credentials

1. Go to [partners.shopify.com](https://partners.shopify.com)
2. Click "Apps" ‚Üí "Create app"
3. Choose "Custom app"
4. Fill in app details:
   - App name: AI-COO
   - App URL: `https://your-app.railway.app`
   - Allowed redirection URLs: `https://your-app.railway.app/auth/callback`
5. Copy API key and API secret to Railway

### Step 5: Deploy

1. Railway will auto-deploy on every git push
2. Monitor deployment in Railway dashboard
3. Check logs for any errors
4. Visit your app URL once deployed

## üîß Local Development

### Prerequisites
- Node 18.x
- npm 9.0+

### Setup

1. **Clone Repository:**
```bash
git clone https://github.com/DevToday-MKhan/AICOO.git
cd AICOO
```

2. **Install Dependencies:**
```bash
npm install
```

3. **Configure Environment:**
```bash
cp .env.example .env
# Edit .env with your credentials
```

4. **Initialize Database:**
```bash
npx prisma migrate dev
npx prisma generate
```

5. **Run Development Server:**
```bash
npm run dev
```

6. **Build for Production:**
```bash
npm run build
npm start
```

## üêõ Troubleshooting

### Build Fails
- Check Node version: `node -v` (should be 18.x)
- Clear node_modules: `rm -rf node_modules && npm install`
- Check TypeScript errors: `npm run typecheck`

### Shopify Auth Fails
- Verify SHOPIFY_APP_URL matches Railway domain
- Check redirect URLs in Shopify Partner dashboard
- Ensure SHOPIFY_API_KEY and SECRET are correct

### GPT Not Responding
- Verify OPENAI_API_KEY is valid
- Check Railway logs for OpenAI errors
- Ensure billing is set up on OpenAI account

### Database Errors
- Verify DATABASE_URL is set
- Run migrations: `npx prisma migrate deploy`
- Check Prisma client generation: `npx prisma generate`
